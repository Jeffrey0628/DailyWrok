- git stash的简单用法

  - 本地开发到一半

  - git stash，缓存一下

  - git stash list，查看

  - git stash apply，将缓存应用，然后就解决冲突就可以了

    - 默认是最上面的

    - ```shell
      git stash apply stash@{1}
      ```

      指定某个缓存
